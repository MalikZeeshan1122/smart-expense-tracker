## 🚀 Features
- **Modern Fintech UI**: Sleek dark mode, glassmorphism gradients, and custom Google Poppins typography.
- **Advanced Dashboard**: Visual insights with native interactive PieCharts powered by `flet-charts`.
- **Smart Categorization (AI)**: Natural language processing engine to extract amounts, dates, and intent with a single string.
- **Local SQLite DB**: Lightning-fast local persistence with zero cloud risk.
- **Budget Pacing**: Visual progress bars alerting you when you exceed arbitrary monthly limits.
- **CSV Data Export**: One-click download of all transaction histories offline.

## 🛠️ Tech Stack
- **Python**: Core logic and internal APIs.
- **Flet Framework**: UI engine backed by Flutter (Cross-platform for Web, Android, iOS, Windows, macOS, Linux).
- **flet-charts**: Dedicated visual diagrams for the dashboard logic.
- **SQLite**: Local relational database.

## 💻 Getting Started
1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/smart-expense-tracker.git
   cd smart-expense-tracker
   ```
2. **Install requirements:**
   ```bash
   pip install flet flet-charts
   ```
3. **Run the tracker:**
   ```bash
   python main.py
   ```
